Git Exercise!!

This is an exercise using git

  1. Create a new public GitHub repository, and name it whatever you like. Be sure to check the box for “initialize this repository with a README.”
  2. Next, clone the repo to your local directory.
  3. Then create an HTML file inside the local directory.
  4. Add, Commit, and Push the code to GitHub.
A Short List of Git Commands
Command Usage Syntax
git clone used to clone a remote repository 'git clone <remote repository URL>'
git add used to add a file to 'git add <file name>'
git commit used to commit a change to the git repository git commit -m "commit message"
git push used to push a commit to a remote repository 'git push [origin master]'
git pull used to pull a commit froma remote repository "git pull"
